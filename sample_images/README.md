# Sample Images

Buraya birkaç test resmi koyun (`test.jpg`, `test.png`, vb.). Workshop boyunca filtreleri bu resimler üzerinde deneyeceksiniz.

## Öneriler

- 600×400 – 1200×800 arasında, manzara veya portre
- Telif hakkı olmayan bir kaynak (Unsplash, Pexels) veya kendi resminiz
- En azından bir tane `test.jpg` zorunlu — README'deki örnekler bu ismi kullanıyor

## Örnek İndirme

```bash
# Unsplash'ten ücretsiz bir test resmi (Creative Commons)
curl -o test.jpg "https://images.unsplash.com/photo-1519681393784-d120267933ba?w=800"
```

> Not: Büyük resim dosyalarını repo'ya commit etmeyin. Gerekirse `.gitignore`'a ekleyin.
